## Patterns

- **BFS / DFS** — Grid traversal, multi-source spread, connected components
- **Topological Sort** — Kahn's (BFS) + DFS-based, DAG cycle detection
- **Union-Find (DSU)** — Path compression + union by rank, connectivity queries
- **Shortest Path** — Dijkstra, Bellman-Ford, Floyd-Warshall
- **Advanced** — Tarjan's bridges, Eulerian path, bipartite coloring
